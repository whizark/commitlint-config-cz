### BREAKING CHANGE

* Now commitlint-config-cz provides only `type-enum`,
`scope-enum` rules by the default.
In case that you need other rules, you may override other config with
commitlint-config-cz rules.

```js
module.exports = {
    extends: [
        'other-config',
        'cz'
    ]
};
```
